	-Works with EAC on or off.
	-All 14 languages supported
	-HUDPlus - BMCounter07Days-KeppDayCount is SAFE to install on new or existing games.
	-HUDPLus - BMCounter07Days-KeppDayCount is SAFE to remove from an existing game.
